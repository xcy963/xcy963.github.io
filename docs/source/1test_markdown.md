## 3. 文本样式

- 普通文本
- *斜体*：`*斜体*` 或 `_斜体_`
- **粗体**：`**粗体**`
- ***粗斜体***：`***粗斜体***`
- ~~删除线~~：`~~删除线~~`
- ==高亮（某些引擎支持）==：`==高亮==`
- `行内代码`：使用反引号包裹
- <u>下划线（HTML 实现）</u>：`<u>下划线</u>`
